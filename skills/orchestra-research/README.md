# Orchestra Research Skills Subset

This directory vendors a thesis-relevant subset of [Orchestra-Research/AI-Research-SKILLs](https://github.com/Orchestra-Research/AI-Research-SKILLs).

Vendored source:

- Repository: `https://github.com/Orchestra-Research/AI-Research-SKILLs`
- Commit: `28f2d29236f2bade2eb92cadb2585189589a5828`
- License: MIT

Included categories:

```text
AI-Research-SKILLs/20-ml-paper-writing
AI-Research-SKILLs/21-research-ideation
AI-Research-SKILLs/22-agent-native-research-artifact
```

Why only this subset:

- `21-research-ideation` helps move from vague idea to defensible thesis direction.
- `20-ml-paper-writing` helps with research-paper structure, citations, tables, plots, and academic writing.
- `22-agent-native-research-artifact` helps package evidence, provenance, and rigor review.

The full upstream repository contains many additional engineering skills for model training, inference, RAG, MLOps, multimodal systems, and more. Those are intentionally not vendored here to keep this thesis template lightweight.

Use `skills/gcc-thesis-template/SKILL.md` as the primary local skill. Treat these Orchestra skills as optional supporting skills for ideation, research writing, plotting, and rigor review.
